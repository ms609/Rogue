RoguePlot Function - Example Output

Branch/Node Coloring Based on Rogue Taxon Attachment Frequencies

What this plot shows:

  • Consensus tree: Built from 9 trees after removing the "rogue" taxon
  • Edge colors: Represent attachment frequencies where the rogue taxon would connect
  • Bright/warm colors: High frequency attachment points (common positions)
  • Dark/cool colors: Low frequency attachment points (rare positions)
  • Legend: Shows the frequency scale from 0.2 to 1.0

Usage Example:

plotted <- RoguePlot(trees, "rogue", legend = "topleft", legend.inset = 0.02) PlotTools::SpectrumLegend( "bottomleft", palette = colorRampPalette(c(par("fg"), "#009E73"), space = "Lab")(100), legend = plotted$legendLabels, cex = 0.4 )
